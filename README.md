> ใช้เซ็นเซอร์ MPU6050 สำหรับอ่านค่ามุมเอียง (Pitch / Roll) แบบเรียลไทม์

> แสดงค่ามุมเอียงผ่าน Serial Monitor เพื่อใช้ในการปรับแต่งระบบ

> มอเตอร์ทำงานสัมพันธ์กับค่ามุมเอียงของหุ่นยนต์โดยตรง

> ใช้ระบบควบคุมแบบ PID (Kp, Ki, Kd) เพื่อรักษาสมดุลของหุ่นยนต์

> สามารถปรับค่า PID เพื่อเพิ่มความเสถียรในการทรงตัว

> หุ่นยนต์สามารถตั้งตรงและรักษาสมดุลได้โดยไม่ต้องมีสิ่งพยุง

> รองรับการควบคุมการเคลื่อนที่ เดินหน้า–ถอยหลัง

> สามารถเลี้ยวซ้าย–ขวา และสั่งงานผ่าน IR Remote
